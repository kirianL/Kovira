# Kovira UI Architecture

## 🎨 Design philosophy

- Minimal SaaS UI
- Card-based system
- High spacing
- Low visual noise

---

## 🧭 Layout system

### App shell

Sidebar + Main content

┌──────────────┬──────────────────────┐
│ Sidebar │ Main Content │
│ Navigation │ Dashboard / Builder │
└──────────────┴──────────────────────┘

---

## 🧩 Core screens

### Dashboard

- stats cards
- activity feed
- quick actions

---

### Form Builder

3-panel layout:

Components | Canvas | Properties

---

### Submissions

- table + detail drawer
- filters
- search

---

### Workflows

- node-based flow builder
- drag & connect system

---

## 🎨 UI system rules

- 8px spacing system
- radius: 10–12px
- soft shadows
- neutral-heavy palette
- single primary color (indigo)
